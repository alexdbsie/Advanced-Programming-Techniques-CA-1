# Task 3 - TCP Client-Server with SQLite

This project is a client-server application that sends customer data over a TCP connection and stores it in a SQLite database.

## Features

- Send customer data from client to server over TCP socket
- Store received data in a SQLite database
- Generate and return a unique registration ID to the client
- Display registration ID on both client and server side

## Files

- server.py - Server logic
- client.py - Client logic
- customers.db - SQLite database
- Screenshots - Output images
