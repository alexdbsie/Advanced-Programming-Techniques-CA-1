# Task 4 - Web Scraper with CSV Output

This project is a Python script that scrapes book data from books.toscrape.com and saves the results to a CSV file.

## Features

- Fetch webpage using the Requests library
- Extract book details (Title, Rating, Price) using BeautifulSoup
- Save extracted data to a CSV file
- Display final output as a formatted table in the terminal

## Files

- Task_4.py - Main logic
- books.csv - Output CSV file
- Screenshots - Output images
