# Task 1 - Churros Ordering System

This project is a console-based application to manage orders for a Churros food stall using object-oriented programming.

## Features

- Place orders by selecting items from the menu
- Calculate total bill based on item price and quantity
- Manage orders using a Queue (FIFO delivery)
- Automatically run unit tests for the billing method on exit

## Files

- Program.cs - Main logic
- Task 1.csproj - Project file
- Screenshots - Output images
